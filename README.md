# K-Aesthetica

A K-beauty journal — honest reviews, routine breakdowns, and the science behind glass skin.

Static single-file site (`index.html`). No build step.

## Local preview
```bash
python3 -m http.server 4321
# then open http://localhost:4321
```

## Deploy
Hosted on Vercel. Pushing to `main` triggers an automatic redeploy.
